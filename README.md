📊 Job Application Tracker:
A simple and efficient **Job Tracker Web App** built using **Python + Streamlit** to help you manage and track your job applications in one place.


🚀 Features:
➕ Add new job applications  
📋 View all applied jobs  
🔍 Filter jobs by status  
✏️ Update application status (Applied / Interview / Rejected / Offer)  
📊 Dashboard with key metrics:
  - Total Applications  
  - Responses  
  - Offers  

🛠️ Tech Stack

- Python  
- Streamlit  
- Pandas  
- CSV (for data storage)
